FAQ

Q: AI features are not working
A: Verify that your AI provider account and API key are correctly configured. Check the connection status in Settings. Errors may occur if your API key has expired or your balance is insufficient.
Q: Telegram bot is not responding
A: Check if the bot token is valid and your internet connection is stable. Conflicts may occur if the bot is running elsewhere simultaneously. Try restarting the app.
Q: The program won't start
A: Ensure you are running Windows 10 or later. Try reinstalling the app and verify write permissions on the installation path.
Q: Are my API keys safe?
A: All credentials are encrypted with Windows DPAPI and stored locally only. Nothing is sent to external servers. Deleting config.json immediately removes all stored credentials.
Q: Pipeline fails during execution
A: Verify that connected external services (Notion, Google Docs, etc.) have valid access permissions. Check the log files for detailed error information.

자주 묻는 질문

Q: AI 기능이 작동하지 않아요
A: AI 제공자 계정과 API 키가 올바르게 설정되었는지 확인하세요. 설정 화면에서 AI 연결 상태를 확인할 수 있습니다. API 키가 만료되었거나 잔액이 부족한 경우에도 오류가 발생할 수 있습니다.
Q: 텔레그램 봇이 응답하지 않아요
A: 봇 토큰이 유효한지, 인터넷 연결이 정상인지 확인하세요. 봇이 다른 곳에서 동시에 실행 중이면 충돌이 발생할 수 있습니다. 앱을 재시작해보세요.
Q: 프로그램이 시작되지 않아요
A: Windows 10 이상인지 확인하세요. 앱을 재설치하거나, 설치 경로에 쓰기 권한이 있는지 확인해보세요.
Q: API 키는 안전한가요?
A: 모든 인증 정보는 Windows DPAPI로 암호화되어 로컬에만 저장됩니다. 외부 서버로 전송되지 않습니다. config.json 파일을 삭제하면 모든 인증 정보를 즉시 제거할 수 있습니다.
Q: 파이프라인 실행 중 오류가 발생해요
A: 연결된 외부 서비스(Notion, Google Docs 등)의 접근 권한이 유효한지 확인하세요. 로그 파일에서 상세한 오류 내용을 확인할 수 있습니다.
