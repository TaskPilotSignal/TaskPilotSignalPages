Roadmap

Upcoming updates after launch (April 2026)

출시 후 업데이트 예정 (2026년 4월 중)

Update 1 — Within 1-2 Weeks

New Pipeline Execution History with Timing

View execution duration and start/end times for each pipeline run directly on the dashboard.

New Telegram Bot Session ID Display

See the active AI session ID in Telegram. Resume the same conversation from your local CLI with --resume.

Improve Global Google Credentials Management

Register Google service accounts once in Settings. Tasks reference them by name instead of per-task file paths.

Improve Modern UI (WinUI Transition)

Rebuilt with WinUI 3 for a faster, smoother, native Windows experience. No more flickering or slow tab switches.

Update 2 — Within 2-3 Weeks

New AI Auto-Generate Pipeline Tasks

Describe what you want in plain language. AI will automatically configure your pipeline sources, analysis, and delivery settings.

New Edit Pipelines via Telegram Bot

Modify pipeline task settings directly from your Telegram AI bot. No need to open the desktop app for quick adjustments.

업데이트 1 — 1~2주 내

New 파이프라인 실행 히스토리 시간 표시

각 파이프라인 실행의 소요 시간과 시작/완료 시각을 대시보드에서 바로 확인할 수 있습니다.

New 텔레그램 봇 세션 ID 표시

텔레그램에서 현재 AI 세션 ID를 확인하고, 로컬 CLI에서 --resume으로 같은 대화를 이어갈 수 있습니다.

Improve Google 인증 글로벌 관리

Google 서비스 계정을 설정에서 한 번 등록하면, 각 작업에서 이름으로 참조합니다.

Improve 현대적 UI (WinUI 전환)

WinUI 3으로 전면 재구축하여 더 빠르고 부드러운 네이티브 Windows 경험을 제공합니다.

업데이트 2 — 2~3주 내

New AI 자동 파이프라인 생성

원하는 작업을 자연어로 설명하면, AI가 소스/분석/배달 설정을 자동으로 구성합니다.

New 텔레그램 봇으로 파이프라인 수정

텔레그램 AI 봇에서 직접 파이프라인 작업 설정을 수정합니다. 간단한 변경은 데스크톱 앱을 열 필요 없습니다.
