Roadmap

Upcoming updates planned within 2 weeks of launch (April 22, 2026)

4/22 출시 후 2주 내 업데이트 예정 항목

v1.1 — UI & Performance

Target: 1-2 weeks after launch

Improve Faster UI Rendering

Smoother tab switching and language changes. No more screen flickering when navigating between pages.

New Pipeline Execution History

See how long each pipeline run took and when it completed, right on the dashboard.

New AI Auto-Generate Pipeline Tasks

Describe what you want in plain language, and AI will automatically configure your pipeline settings.

v1.2 — Telegram Bot Enhancements

Target: 2-3 weeks after launch

New Session ID Sharing

View active AI session IDs in Telegram. Continue conversations seamlessly between Telegram and your local CLI.

Improve Smarter AI Switching

Improved AI provider fallback when rate limits are hit. Conversations continue without interruption.

v1.3 — Settings & Security

Target: 3-4 weeks after launch

Improve Global Google Credentials

Set up Google service accounts once in Settings, then use them across all tasks by name.

Security Enhanced Protection

Stronger path validation, improved log sanitization, and hardened subprocess handling.
