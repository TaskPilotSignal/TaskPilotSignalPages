Privacy Policy

Last updated: 2026-04-09

1. Information We Process

HyungGwangBot processes information that users directly provide:

  • AI provider API keys (Anthropic, Google, etc.)
  • Telegram bot tokens
  • Notion / Google service connection credentials

All information is stored only on the user's local PC, encrypted with Windows DPAPI.

2. External Transmission

Requests are sent only to external services configured by the user:

  • AI providers (Anthropic Claude, Google Gemini, etc.) — AI analysis requests
  • Telegram API — bot message communication
  • Notion API, Google Workspace API — document creation/editing

We do not transmit data to our own servers. HyungGwangBot does not operate any separate servers.

3. Telegram Data

The app communicates with the Telegram API using the user's own bot token. Conversation content and session data are stored locally only.

4. Local Storage

All settings, logs, and session data are stored only in the app's installation directory on the user's PC. Nothing is sent to external servers.

5. Third-Party Services

Each AI provider and external service has its own privacy policy:

  • Anthropic: anthropic.com/privacy
  • Google: policies.google.com/privacy
  • Telegram: telegram.org/privacy

6. Data Deletion

All local data is removed when the app is uninstalled. Deleting config.json immediately removes all stored credentials.

7. Contact

Privacy inquiries: GitHub Issues

개인정보처리방침

최종 수정일: 2026-04-09

1. 수집하는 정보

HyungGwangBot은 사용자가 직접 입력한 다음 정보를 처리합니다:

  • AI 제공자 API 키 (Anthropic, Google 등)
  • 텔레그램 봇 토큰
  • Notion / Google 서비스 연결 정보

이 정보는 모두 사용자의 로컬 PC에만 저장되며, Windows DPAPI로 암호화되어 보호됩니다.

2. 외부 전송

사용자가 설정한 외부 서비스에만 요청을 전송합니다:

  • AI 제공자 (Anthropic Claude, Google Gemini 등) — AI 분석 요청
  • Telegram API — 봇 메시지 송수신
  • Notion API, Google Workspace API — 문서 작성/수정

당사 서버로 데이터를 전송하지 않습니다. HyungGwangBot은 별도의 서버를 운영하지 않습니다.

3. 텔레그램 데이터

사용자의 텔레그램 봇 토큰을 사용하여 Telegram API와 통신합니다. 대화 내용 및 세션 데이터는 로컬에만 저장됩니다.

4. 로컬 저장

모든 설정, 로그, 세션 데이터는 사용자 PC의 앱 설치 경로에만 저장됩니다. 외부 서버로 전송되지 않습니다.

5. 제3자 서비스

각 AI 제공자 및 외부 서비스의 개인정보 처리방침은 해당 서비스의 약관에 따릅니다:

  • Anthropic: anthropic.com/privacy
  • Google: policies.google.com/privacy
  • Telegram: telegram.org/privacy

6. 데이터 삭제

앱 삭제 시 모든 로컬 데이터가 삭제됩니다. config.json 파일을 삭제하면 모든 인증 정보를 즉시 제거할 수 있습니다.

7. 문의

개인정보 관련 문의: GitHub Issues
